About

Super Tools is a collection of small, lightweight web tools for quick calculations, conversions, and mini-helpers, designed for everyday work and for learning by doing. Everything runs directly in the browser (no installations).

Goal

Build simple, clear, and immediate tools: one page = one solved problem. Each tool is designed to be:

  • Fast (minimal UI, instant calculation)
  • Transparent (formula visible, no “magic”)
  • Offline-friendly (once the page is loaded, it works without external services)

How the tools are built

  • Vanilla HTML/CSS/JS (no build step, no dependencies)
  • Validated inputs and clear messages
  • Readable results (sensible formatting)

Notes & disclaimer

These tools are educational / utility. Always verify results when using them in critical contexts (design, safety, standards, regulations, etc.).

If you find a bug or want a feature, open an Issue in the repository.

Privacy

No data is sent to any server: calculations happen in your browser. Hosting is on GitHub Pages.

Contacts / contributions

Want to propose a tool? Great: create a page under /tools/ and add the link on the Home page.
