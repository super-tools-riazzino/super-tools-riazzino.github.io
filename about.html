About

Super Tools è una raccolta di piccoli strumenti web “leggeri” per calcoli rapidi, conversioni e mini-helper pensati per lavoro quotidiano e per imparare facendo. Tutto gira direttamente nel browser (senza installazioni).

Obiettivo

Creare tool semplici, chiari e immediati: una pagina = un problema risolto. Ogni strumento è pensato per essere:

  • Veloce (interfaccia minimale, calcolo istantaneo)
  • Trasparente (formula visibile, niente “magia”)
  • Offline-friendly (una volta caricata la pagina, funziona senza servizi esterni)
Suggerimento: ogni tool vive sotto /tools/ e può essere aggiunto/aggiornato senza impattare gli altri.

Come sono fatti i tool

  • HTML/CSS/JS “vanilla” (niente build, niente dipendenze)
  • Input validati e messaggi chiari
  • Risultati leggibili (formattazione sensata)

Note & responsabilità

Questi strumenti sono educational / utility. Verifica sempre i risultati quando li usi in contesti critici (progettazione, sicurezza, normative, ecc.).

Se trovi un errore o vuoi una feature, apri una Issue sul repository.

Privacy

Non viene inviato nessun dato a server: i calcoli avvengono nel browser. L’hosting è su GitHub Pages.

Contatti / contributi

Vuoi proporre un tool? Ottimo: prepara una pagina in /tools/ e aggiungi il link in Home.
